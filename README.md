## Prerequisites

- Node.js (v14 or higher)
- MongoDB
- npm

## Installation

1. Clone the repository:

```bash
git clone https://github.com/hassan-kamel/fixed-be-task.git
```

```bash
cd fixed-be-task
```

2. Install dependencies:

```bash
npm install
```

## Compile and run the project

### Development mode

```bash
$ npm run start:dev
```

### Production mode

```bash
$ npm run start:prod
```
